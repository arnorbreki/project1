Project 4

CNC out of plywood

For this project I teamed up with Sverrir Kristinsson.

The purpose for this project was for students to get familiar with CNC milling, learn how design for CNC manufacturing and create toolpaths. The goal was to mill something big out of plywood. We wanted to build something that we could use and after some time thinking I remembered that I needed a bookshelf in my room. So we sat down and started the initial design of the bookshelf with in mind what we had learnt from Halfiði's lectures on CNC manufacturing.

Modeling

I began modeling the bookshelf in Solidworks. I choosed Solidworks for this project just to get some practice and becasue I like the asseembly feature of the program. I started by adding parameter for the thickness of the plywood so that I could easily change the design when going to Fablab and CNC it. This is done in Solidworks by adding a global valiable in "Equations" in the tools tab. The design was all about smooth curves that could maybe be hard to saw out of plywood with out the help of CNC machine. I modeled the lower and upper shelves first and then the plates between afterwards. I used some books that I own to decide the size of the shelf so I could fit all my school books in the there. I created some slots for the vertical plates so they would press fit into the shelves and it would be easier to screw them together. These had the depth of 5mm and would be easily graved by the CNC machine. When I was finished I saved my assembly to another file and started to move all the components to one plane so they would be planar for the DFX file. When wanting to save my parts as a DFX file I found that this was hard to do in Solidworks. You can save part as a DFX easily but I needed to google me through how to do it with assemblies. I finally found out that I could save my assembly as a 2D drawing in the scale 1:1 and then save it as a DFX file. Here you can find a video on how to solve this problem.

Prepairing toolpaths in VCarve

For prepairing the model for milling we used VCarve pro, this is a softwere that helps with postproccesing of vector files and calculates toolpaths. When I first uploaded my DFX into VCarve I saw that I needed to add some T-bones in the corners of the press fit slots. The T-bones are created in the corners of the slot because of the circular shape of the mill bit. The mill bit will not reach the corners perfectly so T-bones are added. You can read more about this problem here. Before adding the T-bones in the corners I needed to change some of the vectors of the DFX file. Some of the vector lines were combined and some were not. I right clicked on the lines I wanted to change and there I could press "Node edit mode" and there I selected the lines I wanted to edit, right clicked again and selected "Close vectors with lines". The T-bones were added by selecting the fillet opiton and clicking on the corners of the slots.

Creating toolpaths

After the preparation I opened the toolpaths tab in Vcarve. I started by adding drilling toolpath. The drilling toolpath was used to see if everything was in order and if the bit is in the right place in the start of milling. It is good to start the milling by doing something simple to see if everything is set up the right way. The drilling was only 1mmm of deapth and the idea was to place screws there after the drilling to hold the plywood better in place. Next I added the slot graving. The slot graving had the deapth of 5mm. Then I added the profile toolpath with the deapth of 18.3mm so the bit could mill all the way though the plywood. For all toolpaths I selected 6mm end mill bit and here are the settings for the bit speed rate: Here you can see our toolpath.

The manufacturing

We then moved to Fablab Reykjavík to manufacture our bookshelf. Hafliði helped us setting up the CNC machine. The machine we used was Shop bot and is a big CNC machine that can take up to 3000x1500mm plywood. we got one of those big plywood plates with 18mm thickness. The first thing we did was to move the plate into the Shop bot and fasten it with screws in the corners. We nedded to be careful to place the screws were the mill bit could not reach them. We started with changing the mill bit to 6mm bit. This was done with couple of steps that need to be completed carefully. Here you can watch a simple video that follows those steps. Before starting to mill it is important to open the Shop bot command console and select the spindle warm up routine. This is vital bacause it heats up the bearings in the spindle and prevents them from wearing out. It is important that this is done before setting the z axis to zero position. The spindle should not touch the plywood during warm up and testing. Next we set the zero point of the x and y axis. This was done by moving the bot to the corner of the plywood plate with the keyboard buttons. Next we ran some test run on our toolpaths by moving the saved toolpath from Vcarve to the Shop bot command console before zeroing the z axis. We saw that everything ran well and then we could set the zero point of the z axis. This was done by selecting the Z zero option in "cuts" in the command console. Then we took out metal plate and placed below the mill bit. Then the Z zero cut lowers the bit until it touches the metal plate and saves the values of the z axis as a zero point. After that everything was ready for milling. For more detailed constructions on how to use the shop bot I recomend Hafliði's savety videos on youtube. I recomend if using the shop bot for the first time it is neccessery to have someone that knows the shop bot well and has used it couple of times to help you. All milling tools can be dangerous if not used the right way. All the toolpaths were moved as a SBP file to the Shop bot one by one and milled. Here you can see the graving of the slots. After the milling we cleaned the bot. The milling was not perfect. The profile cut was not deap enough so we needed to use some tools to loosen out parts from the main plate. Then we used sandpaper to remove sharp edges and corners. We finally screwed the parts together. And the bookshelf was ready!
